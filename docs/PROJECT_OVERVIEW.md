# TransitOps

## Project Description

TransitOps is a modern Fleet Management Platform designed to help transport companies manage their complete fleet operations from a centralized dashboard.

The platform allows organizations to manage

- Vehicles
- Drivers
- Trips
- Maintenance
- Fuel Logs
- Expenses
- Analytics

while automatically enforcing business rules to prevent operational mistakes.

---

## Target Users

- Fleet Manager
- Dispatcher
- Safety Officer
- Financial Analyst

Each role has different permissions using RBAC.

---
